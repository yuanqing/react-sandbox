# react-sandbox

> A simple sandbox for hacking on [React](https://facebook.github.io/react/).

## Usage

First, do:

```sh
$ git clone https://github.com/yuanqing/react-sandbox.git
$ cd react-sandbox
$ npm i
$ npm i -g gulp
```

Then, to start hacking on [`app.js`](app.js), do:

```
$ gulp -o
```
